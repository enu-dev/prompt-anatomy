# Prompt Anatomy

プロンプトに構造を与えると、AIは別人になる。

[**▶ デモを試す**](https://example.github.io/prompt-anatomy/)
（↑ 公開後にURLを差し替えてください）

---

## これは何？

ChatGPTやClaudeに頼んでも思った答えが返ってこないとき、
たいていプロンプトに「構造」がありません。

このツールは、プロンプトを **Role / Context / Task / Format** の4要素に分解して、
構造化された強いプロンプトに変換するためのWebツールです。

エンジニアでもエンジニアじゃなくても使えます。

---

## できること

### 💉 診断モード
あなたが普段書いているプロンプトを貼り付けると、
- 4要素の有無を色付きで判定
- 100点満点の品質スコアを表示
- 何が足りないかを具体的にアドバイス
- ワンクリックで構造化バージョンに変換

### 🧱 ビルドモード
ゼロから組み立てる用。
- 5種類のテンプレート（コード生成 / 文章校正 / アイデア発散 / 要約 / 翻訳）
- 4つのブロックを埋めるだけ
- XML形式 / プレーン形式 で出力
- ワンクリックでコピーしてClaudeやChatGPTにそのまま貼り付けられる

---

## 使い方

1. [デモページ](https://example.github.io/prompt-anatomy/) を開く
2. 「診断する」タブに、いつものプロンプトを貼り付け
3. 「診断する」ボタン → スコアと足りない要素が出てくる
4. 「構造化する →」ボタン → 自動的にビルドモードへ
5. 内容を整えて「コピー」→ そのままAIに使う

## 自分で育てるツールです

このツールは「完成品」じゃなくて、**自分の用途に合わせて育てる前提**で作りました。

辞書とテンプレートが分離されているので、`js/` の中身を書き換えるだけで自分専用のプロンプト工房になります。

### 1. テンプレートを追加する

`js/templates.js` の `TEMPLATES` 配列に追加：

```javascript
{
  id: 'my-template',
  name: 'マイテンプレート',
  icon: '🚀',
  role: 'あなたは...',
  context: '前提条件...',
  task: '何をしてほしい...',
  format: '出力形式...'
}
```

### 2. 診断辞書を増やす

`js/diagnose.js` に判定キーワードの辞書があります：

- `ROLE_PROFESSION_PATTERNS`：役職・職種の語彙（自分の業界の専門職を追加）
- `CONTEXT_KEYWORDS`：背景情報を示す語彙
- `TASK_VERBS`：指示動詞の語彙
- `FORMAT_KEYWORDS`：出力形式を示す語彙

たとえばマーケティング業務で使うなら「KPI」「ペルソナ」「ファネル」を `CONTEXT_KEYWORDS` に、
医療業務なら「診療科」「患者」「症例」を入れる、というように業界に合わせて拡張できます。

辞書を育てるほど、自分の文脈に強いプロンプトチェッカーになります。

### 3. 配色やブロックも変えられる

`css/style.css` の冒頭 `:root` でカラー定義をまとめています。
4ブロックの色も変えられるし、ブロック構造自体（Role/Context/Task/Formatを別の枠組みに）変えるのもありです。

---

## なぜ作ったか

AIに頼んでも「うーん」となる回答ばかり返ってくるとき、
プロンプトに **役割（Role）・前提（Context）・指示（Task）・形式（Format）**
の4要素が揃っていないことがほとんどでした。

毎回これを意識して書くのは大変なので、構造そのものを体験できるツールにしました。
AIと壁打ちしながら作っています。
